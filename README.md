# 🧭 Dashboard – User Authentication Screens

This is a responsive,modern dashboard with the following screens:

- 🔓 Login (index.html)
- 📝 Signup
- 🔑 Forgot Password
- 🛠️ Change Password
- 🧭 Dashboard (after login)
- ⚙️ Profile Settings (profile_set.html)
